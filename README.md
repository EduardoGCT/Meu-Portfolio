# Portfólio - Eduardo Castro

Este projeto é o meu site de portfólio pessoal. Ele apresenta quem sou, um resumo das minhas áreas de atuação, projetos em destaque e formas de contato. O objetivo é ter um espaço claro e responsivo para divulgar meu trabalho e facilitar o contato profissional.

## Tecnologias utilizadas

- HTML5 para a estrutura das páginas e seções do portfólio.
- CSS3 para o layout, tipografia e animações.
- JavaScript para interações (menu mobile, destaque do menu ao rolar a página, acordeão de projetos e animações de entrada).
- Font Awesome para ícones.
- Google Fonts (Poppins) para tipografia.

## Versionamento e deploy

- Versionamento mantido com Git.
- Repositório remoto no GitHub.
- Deploy contínuo configurado na Netlify.
