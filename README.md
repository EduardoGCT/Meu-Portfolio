# Portfólio - Eduardo Castro

Este projeto e o meu site de portfolio pessoal. Ele apresenta quem eu sou, um resumo das minhas areas de atuacao, projetos em destaque e formas de contato. O objetivo e ter um espaco claro e responsivo para divulgar meu trabalho e facilitar o contato profissional.

## Tecnologias utilizadas

- HTML5 para a estrutura das paginas e seções do portfolio.
- CSS3 para o layout, tipografia e animacoes.
- JavaScript para interacoes (menu mobile, destaque do menu ao rolar a pagina, acordeao de projetos e animacoes de entrada).
- Font Awesome para icones.
- Google Fonts (Poppins) para tipografia.

## Versionamento e deploy

- Versionamento mantido com Git.
- Repositorio remoto no GitHub.
- Deploy continuo configurado na Netlify.
